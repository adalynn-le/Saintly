Change Log

Stay up to date with all of Saintly's changes and updates! (Also sort of just a diary for me to track progress but yk opensource)

April 24th 2026
Hints and Steps + Daily Problem
In response to past advice, Saintly has added a strike based reveal system that allows you to have multiple tries and see hints/the first few steps before giving up. We also added a Daily Problem system so you can get a new challenge every day
circle
April 15th 2026
Volunteer Application
Saintly cannot function alone! If you want to help out, check out the heart and hand icon in the top right to look at volunteering opportunities
circle
April 14th 2026
ELO system for Adaptive
Adaptive Practice was updated to transform typical, strict question difficulty based sourcing into an adaptive system mirroring the chess ELO system. Questions are generated randomly within range to your own score and adjust in diffculty by range and accuracy.
circle
April 12th 2026
Spaced Repetition + Forms
Created a contact form and a submit a solution form (powered by formspree!) to enable easy communication and allow you to have a part in Saintly! Also added an algorithim for spaced repetition in adaptive practice order to ensure that you are getting enough practices on questions you get wrong. Fixed a few minor mobile issues with the error 404 page and adaptive buttons, and rearranged styling and text.
